Transformer, allows the user to transform XML files using XSLT.

New file extension
XSLT allows to create arbitrary text files from XML. Expecially for RDF files it is important that the new file has a correct extension according to its format
XSLT Template
the XSLT template used to transform the input XML files

The {{page.title}} component takes the input XML files and runs an XSLT transformation on them. It supports multiple input files and it also supports XSLT parameters serialized in RDF, different parameters can be applied to different files based on their file names.
