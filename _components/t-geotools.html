--- short: t-geotools type: transformer color: indigo title: GeoTools description: Provides geographical projection transformation features. ---

Transformer, provides geographical projection transformation features using the GeoTools library.

Input resource type
IRI of the type of entities containing the geocoordinates, e.g. http://www.opengis.net/ont/gml#Point
Predicate with coordinates
IRI of the predicate connecting the entity of the input resource type to the literal containing the geocoordinates, e.g. the http://www.opengis.net/ont/gml#pos property.
Predicate with coordinates type
IRI of the predicate connecting the entity containing geocoordinates to a literal containing the source coordinates type, e.g. http://www.opengis.net/ont/gml#srsName
Default coordinates type
Coordinates type to be used with entities, which do not specify the coordinates type themselves, e.g. EPSG:5514
Predicate used to connect output entity with
The new entity containing the transformed geocoordinates will be connected to the original entity using this predicate
Type of output coordinates
Target coordinates type, e.g. EPSG:4326
Fail on error
Stops execution on error when enabled, continues execution when disabled.
