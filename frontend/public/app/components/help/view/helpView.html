Issues

Bugs, errors and features requests can be reported as issues on the project GiHub page. If you want to report a bug or an error please include the used version of LinkedPipes ETL (see section Version bellow).

In case of a problem with the web client, it would be appreciated if you could check the browser developer console for any errors, and provide their list as a part of the issue description.

If your are experiencing problem with a certain pipeline and the pipeline (or its modification) is not private. Please feel free to upload the pipeline to the demo instance and provide the link to the pipeline (execution) in the issue description.
